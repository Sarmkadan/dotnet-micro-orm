# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately via GitHub's [Security Advisory](https://github.com/rocksdanister/dotnet-micro-orm/security/advisories) feature. This ensures the vulnerability is handled confidentially.

Alternatively, you may contact the maintainer directly at:
- Email: rutova2@gmail.com
- Response time: Within 48 hours

## Supported Versions

| Version | Support Level |
|---------|---------------|
| v2.0.x  | Full support (security + bug fixes) |
| v1.x    | Security fixes only |

## Disclosure Process

1. **Report**: Submit via GitHub Security Advisory or email
2. **Acknowledgment**: You'll receive confirmation within 48 hours
3. **Assessment**: The maintainer will assess the vulnerability
4. **Fix**: A patch will be released as soon as possible
5. **Public disclosure**: After fix is released (typically within 7-14 days)

## Security Contact

For all security-related matters, please use the GitHub Security Advisory feature or email rutova2@gmail.com. Please do not open public issues for security vulnerabilities.
